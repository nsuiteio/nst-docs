# コントラクトメソッドの実行手順

**【 N Board操作手順 】**

**1：画面左側 サイドメニュー 「 ワークフロー 」 を押下**

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-05 115207.png" alt="" width="282"><figcaption></figcaption></figure></div>

**2：画面右上  新規申請 ≫ コントラクトメソッド ≫ 通常実行の順で選択**

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-04 122720.png" alt="" width="278"><figcaption></figcaption></figure></div>

**3：コントラクトメソッドの申請内容を入力する**

　　┗ ※申請タイトルは必須条件です。

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-05 120631.png" alt="" width="231"><figcaption></figcaption></figure></div>

▼各種申請事項入力後の画面

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-05 122205.png" alt="" width="252"><figcaption></figcaption></figure></div>

**5：承認先プロセスの選択**

　　┗ 登録されている承認先プロセスを選択し、「申請」 を押下

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-04 124925.png" alt="" width="310"><figcaption></figcaption></figure></div>

**6：承認の実施（申請内容の実行）**

　　┗ 指定された承認者の 「承認待ち」 の欄より、内容を確認して 「実行」 を押下

<div align="left"><figure><img src="../../.gitbook/assets/スクリーンショット 2025-06-05 125906.png" alt="" width="375"><figcaption></figcaption></figure></div>
