# スマートフォンアプリについて

{% hint style="info" %}
こちらの記事では、N Suiteのスマートフォン版アプリを使用する際の必要情報をまとめています。
{% endhint %}

{% hint style="info" %}
アプリ版でできること\
・ワークフローでの申請の承認　　・各種署名（トランザクション・メッセージ）\
\
■サポートOSバージョン（Apple製品）\
iPhone：iOS 15.1以降\
\
■サポートOSバージョン（Android製品）\
Android 7.0以降\
\
※注意：サポートのOSバージョンは、アップデートにより変更される場合がございます。
{% endhint %}

■App ダウンロード

Apple：[https://apps.apple.com/us/app/n-suite-app/id6451965451](https://apps.apple.com/us/app/n-suite-app/id6451965451)

Android：[https://play.google.com/store/apps/details?id=io.nsuite.app\&pli=1](https://play.google.com/store/apps/details?id=io.nsuite.app\&pli=1)

■App版、ログインについて\
ログイン方法は、以下の方法でご利用いただけます。

* N Suiteのアカウントを使ってログイン
* Googleアカウントを使ってログイン
* Appleアカウントを使ってログイン

※Appleアカウントでログインする際の注意点

Appleアカウントを使ってN Suiteの新規アカウントを作成しログインする際は、メールアドレスの共有/非公開を指定する必要があります。\
この際、必ず 「メールを共有」 を選択してください。　「メールを非公開」 を選択した場合、ユーザーの Appleアカウント がN Suite側で認識されず、N Suite にアクセスすることができません。
