# MPC署名権の付与

#### Step 1: MPC署名権の付与の開始（PC（N Board）で操作） <a href="#step-1-mpcnonopcn-boardde" id="step-1-mpcnonopcn-boardde"></a>

{% hint style="info" %}
このStepで操作するのはMPC署名権を付与するユーザーです。\
この操作を行うユーザーは下記の条件を満たしている必要があります。

&#x20;\- 組織のオーナーまたは管理者

&#x20;\- MPC署名権を持っている
{% endhint %}

* 組織の「メンバー」タブで、MPC署名権を付与する対象のユーザーのアクションボタンをクリックし、「MPC署名権を付与」をクリックする

#### Step 5: MPC署名権の付与の完了確認（PC（N Board）で操作） <a href="#step-5-mpcnonopcn-boardde" id="step-5-mpcnonopcn-boardde"></a>

{% hint style="info" %}
このStepを操作するのは、MPC署名権を付与するユーザーです
{% endhint %}

* 組織の「メンバー」タブでMPC署名権を付与したユーザーの「MPC署名権」欄にチェックマークが付いていることを確認（チェックマークが付いていればMPC署名権の付与が完了）
