# MPCのアドレス作成

## 操作手順 <a href="#instructions-to-use-article-templates" id="instructions-to-use-article-templates"></a>

#### Step 1: MPCのアドレス作成（PC（N Board）で操作） <a href="#step-1-access-the-template-list-view" id="step-1-access-the-template-list-view"></a>

{% hint style="info" %}

{% endhint %}

*   サイドメニューでアドレスを作成するチームをクリック

<figure><img src="../../../.gitbook/assets/image (126).png" alt=""><figcaption></figcaption></figure>

    <div align="left"><figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/1070013260926/original/OHeUpkjzEPEdAM2AWvTJK1f33i5sJDQkjQ.png?1729131254" alt="" width="375"><figcaption></figcaption></figure></div>

<figure><img src="../../../.gitbook/assets/image (127).png" alt=""><figcaption></figcaption></figure>
