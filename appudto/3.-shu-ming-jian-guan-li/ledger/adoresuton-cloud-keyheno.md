# アドレスとN Cloud Keyへの名前登録

**① アドレスへの名前登録**

<figure><img src="../../../.gitbook/assets/a (2).png" alt=""><figcaption></figcaption></figure>

上記のように"Main Wallet" "Sub Wallet"という風に名前を付けることができます。アドレスの右側に表示されている「･･･」から「名前設定」を選択し、任意の名前を入力ください。



**② N Cloud Keyへの名前登録**

<figure><img src="../../../.gitbook/assets/b (2).png" alt=""><figcaption></figcaption></figure>

既存のN Cloud Keyに名前を登録することができます。N Cloud Keyにアクセスするには、右上のアイコンよりアドレス一覧を開き、下の方にある「N Cloud Key 管理」を開きます。

<figure><img src="../../../.gitbook/assets/c (2).png" alt=""><figcaption></figcaption></figure>

N Cloud Key の一覧が表示されますので、名前を登録したいアドレスの右側にある「･･･」から「名前設定」を選択し、任意の名前を入力ください。保存後は上記のように"AWS Test Keys" という風に名前を付けることができます。
