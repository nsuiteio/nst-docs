# チームへの保有アドレス登録

{% hint style="info" %}
**本手順を実施する方**

下記両方の条件に該当するユーザー

* チームの管理者権限（オーナーまたは管理者）が付与さているたユーザー
* N Walletを使用するユーザー
{% endhint %}

{% hint style="warning" %}
本手順は、ブラウザにN Walletをインストールし、事前にN Walletにアドレスを登録した状態で行ってください。
{% endhint %}



① 画面左のパネルで、アドレスを登録するチームをクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-04 at 14.50.29 2.png>)

② チーム画面で「アドレス」タブを選択し、「アドレス登録」ボタンをクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-04 at 14.50.29.png>)

③ プルダウンで「保有アドレス」をクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-04 at 14.50.34.png>)

④ アドレス選択モーダルで、登録するアドレスをクリック

{% hint style="info" %}
ブラウザにインストール済みのN Walletに登録されているアドレスがモーダルに表示されます。
{% endhint %}

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-04 at 14.50.57.png>)

⑤ 登録するアドレスの名前を入力し、「登録」ボタンをクリック

{% hint style="info" %}
アドレスの名前は任意の名前を設定してください。
{% endhint %}

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-04 at 14.51.15.png>)

{% hint style="info" %}
アドレス登録画面にてLedgerがうまく認識されない場合、下記をご確認ください：

・Ledger Liveを起動していないこと

・Ledgerのデバイスでイーサリアムアプリを起動している
{% endhint %}
