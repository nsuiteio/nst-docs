# 組織へのメンバー追加

{% hint style="info" %}
**本手順を実施する方**

組織の管理者権限（オーナーまたは管理者）が付与されているユーザー
{% endhint %}

{% hint style="warning" %}
本手順は、N Boardにログインし、操作を行なってください。\
【N BoardのURL】\
[http://board.nsuite.io/](http://board.nsuite.io/)
{% endhint %}

#### 1. 画面左のパネルで、組織をクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-16 at 21.29.51.png>)

#### 2. 組織画面で「メンバー」タブを選択し、「メンバー追加」ボタンをクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-16 at 21.32.12.png>)

#### 3. メンバー追加モーダルで、追加するメンバーの「メールアドレス」と「ロール」を指定し、「追加」ボタンをクリック

![](<../../../../../.gitbook/assets/Screen Shot 2022-02-16 at 21.30.32.png>)

組織ロールの詳細については下記ページを参照ください：

[組織ロールの説明](../rru/rruno.md)
