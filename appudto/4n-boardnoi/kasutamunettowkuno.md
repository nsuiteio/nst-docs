# カスタムネットワークの追加

## カスタムネットワークの追加方法 <a href="#instructions-to-use-article-templates" id="instructions-to-use-article-templates"></a>

① N board上でネットワークを追加

1. チームを選択
2. ネットワークのタブを選択
3. ネットワークの追加をクリック
4. ポップアップウィンドウの各フィールドに追加するネットワークの情報を入力し「追加」をクリック
