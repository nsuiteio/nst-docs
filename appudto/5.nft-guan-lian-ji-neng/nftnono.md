# NFTの送付の方法

① N Suiteに保有しているNFTのコントラクトを登録する。

1. 該当のチームを選択する。
2. コントラクトのタブをクリックする。
3. コントラクト登録をクリックする。
4. NFTをクリックする。
5. NFTの追加のウィンドウに必要な情報を入れて追加をクリックする。

<figure><img src="../../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
現在、N SuiteではNFTの表示機能がないため、N Suiteの画面では保有しているNFTを閲覧することはできません。外部のサイトで保有しているNFTの情報を確認してください。
{% endhint %}

② コントラクトメソッドを申請する。

1. ワークフローをクリック
2. 新規申請をクリック
3. コントラクトメソッドをクリック
4. 通常実行をクリック
5. 実行するチームを選択
6. コントラクトの申請画面に下記必要情報を入力して申請をクリック

* ネットワーク：① で登録したネットワーク
* 実行アドレス：実行するアドレス
* コントラクト：①で登録したコントラクト
* メソッド：safeTrasferFrom

<figure><img src="../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>

③ N Boardワークフローの承認プロセスを通して署名実行する。

1. 承認者がN Board上で承認
2. 実行者がN Board上で実行

{% hint style="warning" %}
送付先のアドレスがN Suiteに登録している別のアドレス（保有アドレス）の場合でも手順は同じです。
{% endhint %}
