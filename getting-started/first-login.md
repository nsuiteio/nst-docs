# 初回ログイン手順

{% hint style="info" %}
**本手順を実施する方**

① N Suiteのご利用申し込み後、担当者より環境開設の完了連絡があった方、かつ

② 初期設定のシートにオーナーとして登録された方（下記手順①を参照）
{% endhint %}

### ① オーナーの確認

本手順を実施できるのはN Suiteの担当者へ提出された「初期設定」スプレッドシートにオーナーとして登録されたユーザーのみです。アドレスも送って頂いたものを使用しますので、ログインできる端末を使用していることを事前にご確認ください。

![](<../.gitbook/assets/image (1).png>)

### ② N BoardのURLへアクセス

オーナーの端末で [https://board.nsuite.io/](https://board.nsuite.io/) へアクセス。下記画面が表示されます。

![](<../.gitbook/assets/image (2).png>)

### ③ サインアップへ進む

ログインボタンの下に表示されている「サインアップ」の文字列をクリックして進みます。

![](<../.gitbook/assets/image (3).png>)

### ④ 必要事項を記入後、「アカウント作成」をクリック

メールアドレスはスプレッドシートで記入し送付したアドレスと同一のものを入力。パスワードはセキュリティ要件を満たすもの入力し、「アカウント作成」で進みます。この際、利用規約とプライバシーポリシーの確認もお願いいたします。

![](<../.gitbook/assets/image (4).png>)

### ⑤ メールアドレス宛に届いた認証メールを確認

入力したメールアドレスの受信箱をご確認ください。no-reply@nsuite.io というアドレスから自動配信メールが届いていることを確認し、認証のリンクをクリックし登録を完了させます。

![](<../.gitbook/assets/image (5).png>)

### ⑥ログイン確認

再度 [https://board.nsuite.io/](https://board.nsuite.io/) へアクセスし、本手順で設定した情報にてログインが可能になっていることを確認します。
