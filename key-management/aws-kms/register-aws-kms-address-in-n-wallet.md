# N Walletの初期設定（AWS KMSのアドレス登録）

{% hint style="info" %}
AWS KMSで秘密鍵を管理している場合、本手順でN Walletにアドレスを登録してください。
{% endhint %}

{% hint style="info" %}
本手順を行う前に、AWSで秘密鍵の作成を行うご担当者に下記の作業を行ってもらい、AWSの認証情報（`Access Key ID` と`Secret Access Key` ）を入手してください。詳細は秘密鍵の管理者向け手順を確認してください。
{% endhint %}

① 秘密鍵管理サービスでAmazon Web Serviceを選択

![](<../../.gitbook/assets/Screen Shot 2021-12-10 at 18.17.59 (1).png>)

② AWSの認証情報（`Access Key ID` と`Secret Access Key` ）を入力して、「登録」ボタンをクリック

![](<../../.gitbook/assets/Screen Shot 2021-12-10 at 18.18.03 (1).png>)
