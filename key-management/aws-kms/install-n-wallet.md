# N Walletのインストール

{% hint style="info" %}
本手順は、Google Chromeから行ってください。
{% endhint %}

{% hint style="warning" %}
お使いのGoogle ChromeにMetaMaskがインストールされている場合、N Walletが正しく動作しません。\
MetaMaskがインストールされている場合は、新しくGoogle Chromeのプロファイルを作成し、作成したプロファイルに切り替えて、N Walletをインストールしてください。

**プロファイルの作成手順**

[https://support.google.com/a/users/answer/9310144?hl=ja](https://support.google.com/a/users/answer/9310144?hl=ja)
{% endhint %}



① Google Chromeにて、Chrome ウェブストアのN Walletページにアクセス

**N Walletページ**

[https://chrome.google.com/webstore/detail/n-wallet/fmpjabbciljgnieepgfiikeobnakdhcg?hl=ja\&authuser=0](https://chrome.google.com/webstore/detail/n-wallet/fmpjabbciljgnieepgfiikeobnakdhcg?hl=ja\&authuser=0)



② 「Chrome に追加」ボタンをクリック
