# N Walletの手動アップデート

{% hint style="warning" %}
**手動アップデートが必要なケース**\
N Walletは、自動でアップデートされるため、基本的には手動でアップデートを行う必要はありません。\
しかし、N Walletの新しいバージョンがリリースされてから自動でアップデートされるまでの間にタイムラグがある場合があります。

また、N Walletの旧バージョンがN Suite の新バージョンと互換性がない場合、旧バージョンのN Walletをご利用いただけなくなる場合があります。

そのようなケースで、すぐにN Walletをご利用する場合は手動で新しいバージョンにアップデートしてください。
{% endhint %}



① Chromeの画面右上のアイコンをクリックし、メニューを表示

<figure><img src="../../.gitbook/assets/スクリーンショット 2023-08-29 19.03.12.png" alt=""><figcaption></figcaption></figure>

③ Chromeの拡張機能の管理画面で、「デベロッパーモード」をオンに変更
