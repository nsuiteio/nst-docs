# 任意のトークン追加



{% hint style="info" %}
**本手順を実施する方**

チームのメンバーになっているユーザー
{% endhint %}

{% hint style="info" %}
任意のトークンをチームに追加すると、登録したトークンの送金が行えるようになります。
{% endhint %}

#### **操作手順**

① トークンを追加するチームをクリック

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.50.28.png>)

② チーム画面で、「コントラクト」タブをクリック

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.50.32.png>)

③ 「コントラクト登録」ボタンをクリック

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.50.36.png>)

④ プルダウンで「トークン」をクリック

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.50.39.png>)

⑤ 「トークンの追加」モーダルで、ネットワークを指定し、トークンアドレスを入力した後、「追加」ボタンをクリック

{% hint style="info" %}
トークンシンボルとトークン桁数（decimal）は、トークンアドレスを入力すると自動的に入力されます。
{% endhint %}

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.55.39.png>)

⑥ チームのコントラクト一覧に追加したトークンが表示される

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.55.48.png>)

{% hint style="info" %}
トークン追加が終わると、送金時の「トークン選択」モーダルの「カスタム」タブで、追加したトークンを選べるようになります。
{% endhint %}

![](<../.gitbook/assets/Screen Shot 2022-04-14 at 16.56.01.png>)
