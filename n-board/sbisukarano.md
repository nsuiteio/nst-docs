# 外部サービスからの申請作成

{% hint style="info" %}
**本手順を実施する方**

N Walletをインストールし、アドレスを登録されている方
{% endhint %}

{% hint style="info" %}
Google ChromeにN Walletをインストールし、アドレスを登録した状態で、OpenSeaやUniswap等のサービスで、トランザクションへの署名が必要な操作を行うと、N Boardで申請を作成することができます。\
申請が承認された後、トランザクションに署名し、操作を実行することができます。
{% endhint %}

**対応サービス**

本機能は、MetaMaskが対応しているサービスで利用可能です。\
ただし、動作検証が取れているサービスおよび操作は下記の通りです。

## **操作手順**

① 外部サービスでN Walletを接続

{% hint style="info" %}
サービスのウォレット接続画面で、接続するウォレットを選択する必要がある場合、「MetaMask」を選択してください。
{% endhint %}

例）OpenSeaでN Walletを接続する場合

![](<../.gitbook/assets/Screen Shot 2022-03-07 at 21.10.28.png>)

② 外部サービスでトランザクションに必要な操作を実行

例）OpenSeaでNFTを購入をする場合

NFTで購入操作を実行

![](<../.gitbook/assets/Screen Shot 2022-03-07 at 20.50.22.png>)

③ N Walletのポップアップが表示されるため、「申請」ボタンをクリック

![](<../.gitbook/assets/Screen Shot 2022-03-07 at 20.50.31.png>)

④ N Boardのチーム選択モーダルが表示されるため、申請を作成するチームを選択

{% hint style="warning" %}
所属しているチームが一つの場合は、チーム選択モーダルは表示されず、手順⑤の画面が表示されます。
{% endhint %}

![](<../.gitbook/assets/Screen Shot 2022-03-07 at 20.53.30.png>)

⑤ N Boardで申請フォームが表示されるため、必要な項目を入力し、「申請」ボタンをクリック

{% hint style="info" %}
「実行内容」の項目は入力された状態で申請フォームが作成されます。

申請タイトルや申請メッセージなど「実行内容」以外の項目を入力して、「申請」ボタンをクリックしてください。
{% endhint %}
