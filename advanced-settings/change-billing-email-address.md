# 請求書送付先メールアドレスの変更手順

{% hint style="info" %}
※事前に N Board へログインしてください。
{% endhint %}

**【 請求書送付先 メールアドレスの変更 操作手順 】**

**1：**&#x753B;面左側の 「組織」 を押下

2：画面中央側 「設定」 タブを押下

3：「請求先メールアドレス」 欄のアドレスを、ご希望のアドレスへ変更

4：「保存」 を押下
