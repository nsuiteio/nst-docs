# スマホUI: 申請一覧と申請詳細

{% hint style="info" %}
モバイル版のN Suiteは2022年の9月時点で存在しませんが、スマホのブラウザより操作可能な箇所を記載しています。
{% endhint %}

## 概要

N Board ([board.nsuite.io](https://board.nsuite.io/)) にスマホブラウザ (Chrome, Safari, Brave等) を使ってログインすると、スマホUIが表示されます。また、申請詳細を開いてからPC版のUI同様に承認、却下の作業も可能です。

② 申請詳細を開く ＋ 承認作業

![](<../.gitbook/assets/image (54).png>)

詳細を開きたい申請をタップし、承認や実行権限がある申請については「却下」「承認」等のボタンが表示されていることをご確認ください。PC版のUI同様に作業をスマホから行うことができます。
