# N Suiteドキュメント

N Suiteの利用開始、管理者設定、署名・鍵管理、N Boardの操作、応用機能に関する顧客向けドキュメントです。

## 初めて利用する方へ

初回利用時は、以下の順に確認してください。

1. [初回ログイン手順](getting-started/first-login.md)
2. [動作環境](getting-started/operating-environment.md)
3. [導入・初期設定](setup/README.md)
4. [署名・鍵管理](key-management/README.md)

## 目的別の入口

* 組織、チーム、ロール、ワークフローを設定する場合は[導入・初期設定](setup/README.md)を確認してください。
* 署名方式やN Walletの設定を確認する場合は[署名・鍵管理](key-management/README.md)を確認してください。
* 申請作成やN Board上の操作を確認する場合は[N Boardの使い方](n-board/README.md)を確認してください。
* NFT、コントラクト実行、Wallet Connectなどの機能を確認する場合は[NFT関連機能](nft/README.md)または[応用機能・各種設定](advanced-settings/README.md)を確認してください。
