# N Suiteドキュメント

N Suiteの利用開始、管理者設定、署名・鍵管理、N Boardの操作、応用機能に関する顧客向けドキュメントです。

## 初めて利用する方へ

初回利用時は、以下の順に確認してください。

1. [初回ログイン手順](appudto/1hajimeni/roguin.md)
2. [動作環境](appudto/1hajimeni/dong-zuo-huan-jing.md)
3. [導入・初期設定](appudto/2matome/README.md)
4. [署名・鍵管理](appudto/3.-shu-ming-jian-guan-li/README.md)

## 目的別の入口

* 組織、チーム、ロール、ワークフローを設定する場合は[導入・初期設定](appudto/2matome/README.md)を確認してください。
* 署名方式やN Walletの設定を確認する場合は[署名・鍵管理](appudto/3.-shu-ming-jian-guan-li/README.md)を確認してください。
* 申請作成やN Board上の操作を確認する場合は[N Boardの使い方](appudto/4n-boardnoi/README.md)を確認してください。
* NFT、コントラクト実行、Wallet Connectなどの機能を確認する場合は[NFT関連機能](appudto/5.nft-guan-lian-ji-neng/README.md)または[応用機能・各種設定](appudto/6yokuaru/README.md)を確認してください。
